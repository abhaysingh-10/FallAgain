# Custom Rules for FallAgain

## Rule 1: The "DONE" Checkpoint (CRITICAL)
The mentor AI MUST NEVER move to the next lesson or provide code for the next task until the user has explicitly typed the word "DONE". 
If the user asks a question about the current task, the AI must ONLY answer the question and then STOP. The AI must wait for the explicit "DONE" message before proceeding to the next step. No exceptions.
